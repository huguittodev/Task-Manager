**Task Master Pro** no es la típica lista de tareas. Es una aplicación web moderna diseñada para la gestión eficiente del tiempo y las prioridades, construida con las mejores prácticas del desarrollo Frontend actual.

> **💡 Nota:** Puedes ver el proyecto en vivo aquí: [https://task-manager-brown-seven-65.vercel.app/]

![CAP TASK MANAGER](/src/assets/CAP%20TASK%20MANAGER.png)

---

## ✨ Características Principales

- **Gestión Completa (CRUD):** Crea, lee, edita y elimina tareas de forma fluida.
- **Edición en Línea Inteligente:** Modifica tareas directamente en la lista utilizando `FormData` nativo para un rendimiento óptimo, sin necesidad de modales intrusivos.
- **Sistema de Prioridades:** Clasifica tus tareas en Alta (🔴), Media (🟡) o Baja (🟢) con estilos visuales dinámicos.
- **Filtros Avanzados (Estados Derivados):** Filtra simultáneamente por estado (completadas/pendientes) y por prioridad sin perder el estado original de los datos.
- **Ordenación Dinámica:** Ordena tus metas por las más recientes o las más antiguas con un solo clic.
- **Persistencia de Datos:** Tus tareas nunca se pierden. El sistema guarda automáticamente todo en el `localStorage` del navegador.
- **UI/UX Premium:** Interfaz oscura, minimalista, con efectos de _glassmorphism_ (cristal), transiciones suaves y diseño 100% responsivo.

## 🛠️ Tecnologías Utilizadas

Este proyecto ha sido desarrollado utilizando un stack moderno para garantizar velocidad, tipado estricto y escalabilidad:

- **[React 18](https://react.dev/):** Librería principal para la interfaz de usuario.
- **[TypeScript](https://www.typescriptlang.org/):** Tipado estricto para evitar errores en tiempo de ejecución (Interfaces y Tipos personalizados).
- **[Tailwind CSS](https://tailwindcss.com/):** Framework de utilidades para un diseño rápido y mantenible.
- **[Vite](https://vitejs.dev/):** Empaquetador extremadamente rápido para el entorno de desarrollo.
- **pnpm:** Gestor de paquetes eficiente.
